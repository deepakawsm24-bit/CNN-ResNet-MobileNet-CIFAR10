# CNN Image Classification (ResNet18 vs MobileNetV2)
# Overview 
This project implements image classification using Convolutional Neural Networks (CNN). Two models, ResNet18 and MobileNetV2 , are trained and evaluated on the CIFAR-10 dataset.
# Objective
- Implement CNN-based image classification
- Compare ResNet-18 and MobileNetV2 performance
- Analyze accuracy and efficiency
# Dataset
- CIFAR - 10
- 60,000 images
- 10 classes
# Technologies Used
- Python
- PyTorch
- Google Colab
# Models Used
- ResNet18
- MobileNetV2
# Results
 |    Model    |    Accuracy  |
 | ResNet 18   |    87.6%     |
 |MobileNetV2  |    89.7%     |
## How to Run
1. Open the notebook in Google Colab
2. Run all cells
3. Train and evaluate the model
# Conclusion
MobileNetV2 performed better with higher accuracy and efficiency compared to ResNet18.
